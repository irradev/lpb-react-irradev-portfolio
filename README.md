# irraDev Portfolio

Este repositorio contiene **mi portafolio personal como Frontend Engineer**.
Funciona como un **laboratorio de desarrollo**, donde documento y publico prácticas profesionales, pruebas técnicas y algunos proyectos reales desarrollados para clientes.

El objetivo no es solo mostrar resultados visuales, sino **criterio técnico**, calidad de código, experimentación con tecnologías frontend y evolución profesional.

### Tecnologías principales

- Vue 3 (Composition API)
- React
- Testing (Vitest, Vue Test Utils)
- CSS puro y Tailwind
- Vite
- JavaScript / TypeScript
- Flutter (en proyectos específicos)

### Enfoque del portafolio

- Prácticas profesionales y ejercicios técnicos
- Pruebas técnicas reales
- Experimentos de UI / UX
- Arquitectura y organización de código

### Aprendizaje continuo y documentación

Algunos proyectos son experimentales; otros fueron desarrollados bajo requerimientos reales.

### Instalación

Instala las dependencias del proyecto:

```bash
yarn
```

### Desarrollo

Ejecuta la aplicación en modo desarrollo:

```bash
yarn dev
```

Ejecuta la aplicación y permite el acceso desde otros dispositivos en la misma red (móvil, tablet, etc.):

```bash
yarn dev --host
```

### Nota

Este portafolio está en constante evolución.
Los proyectos se actualizan, refactorizan o reemplazan conforme profundizo en nuevas tecnologías y mejores prácticas.
